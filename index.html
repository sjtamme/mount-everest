About Mount Everest

Mount Everest, sitting on the border of Nepal and Tibet, is the tallest mountain in the world. The summit sits at 29,035 feet. Although 17 trails have been traveled to the summit of Mount Everest, almost everyone makes the climb via one of two routes, the South Col route starting in Nepal or the Northeast Ridge route starting in Tibet. Each route has a base camp, making up the two base camps on the mountain. More camps along the trail allow climbers to adjust to the high altitude while ascending, but some visitors just hike to a base camp and don't ascend any further. You can "tour" an Everest base camp on National Geographic's website.

Data used in this project

Mount Everest location point extracted from Nepal's country file published by the GEONet Names Server (GNS). Data accessed December, 2020. Contours were extracted from a STRM 1 arc-second global raster digital elevation model. This data can be accessed from Earth Explorer.

Tools used in this project

  • PostGIS was used to store/query Nepal's dataset.
  • Data is shown on Mapbox web map with geolocation services enabled. If the map got popular it would require a fee to use.
  • GDAL and QGIS 3.16 to convert raster elevation units and create contours.
